# 路线图

## P0：15 镜 Pilot

目标：验证即梦官网能否生成非遗纪录片可用镜头。

任务：

- 建立 3 个非遗主题。
- 为每个主题设计 5 个镜头。
- 输出即梦官网可复制提示词。
- 人工生成视频。
- 回传结果。
- 按评分表评估。
- 总结模型可用边界。

交付：

- 15 条镜头提示词。
- 15 条生成记录。
- 评分表。
- 可用镜头类型结论。

## P1：前期制片台

目标：把人工 pilot 中有效的方法做成系统化流程。

任务：

- Project / Episode / Shot / ShotAsset 基础模型。
- 内容智能体。
- 视觉智能体。
- 质检智能体。
- manual_jimeng 执行记录。
- 提示词版本管理。
- 资产上传与绑定。

## P2：素材库与纪录片混合工作流

目标：支持 AI 镜头和真实素材混用。

任务：

- 真实素材资产库。
- 版权字段。
- 事实卡片。
- 人类专家审核状态。
- 真实素材占位镜头。
- 分镜到剪辑清单导出。

## P3：自动化实验

目标：在可控范围内尝试减少人工操作。

任务：

- jimeng_rpa_pilot 浏览器自动化实验。
- 第三方中转 API 稳定性测试。
- 官方 API 权限获得后的 provider 接入。

## P4：成片辅助

目标：辅助剪辑，不承诺完全自动剪纪录片。

任务：

- 字幕。
- 旁白音频。
- 音乐建议。
- 镜头排序建议。
- 剪辑工程导出。
- 简单预览合成。

## 当前优先级

当前只执行 P0。P0 完成前，不进入复杂工程搭建。
