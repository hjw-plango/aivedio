# 非遗纪录片 Pilot

## 目的

P0 不先做完整工程流水线，而是先验证即梦官网能否生成可用的非遗纪录片镜头。

执行方式：

```text
3 个主题
每个主题 5 个镜头
总计 15 条提示词
用户复制到即梦官网手动生成
每条生成 1-3 个版本
下载回传
人工评分
总结可用镜头类型
```

## 15 个镜头

景德镇制瓷：

1. 老作坊外景空镜：清晨自然光，旧砖墙、陶土、木架、窑口烟气。
2. 拉坯手部特写：湿润瓷泥在转盘上成形，手指有泥水和细微颤动。
3. 修坯工具特写：金属工具轻刮瓷坯边缘，粉尘落下。
4. 青花绘制特写：毛笔蘸青料画纹样，瓷面有吸水痕迹。
5. 窑火意象镜头：火光、烟气、窑砖、器物轮廓，不出现具体人脸。

苏绣：

1. 老绣架空镜：窗边自然光，布面、线轴、木绷架、旧剪刀。
2. 针脚特写：针穿过绸缎，丝线反光，手部动作缓慢。
3. 理线镜头：不同颜色丝线被手指分开。
4. 纹样局部动效：花鸟纹样从针脚中逐渐显现，保持纪录片克制。
5. 工坊环境镜头：桌面略杂乱，布料、图稿、线团自然摆放。

川剧变脸：

1. 戏台后台空镜：旧木桌、脸谱、戏服、镜子、暖色灯。
2. 脸谱绘制特写：画笔在面具或脸谱纸样上落色，不做真人正脸。
3. 戏服纹理特写：刺绣、布料、肩甲、袖口随手移动。
4. 舞台剪影：演员背影或侧影甩袖，不展示可识别正脸。
5. 变脸意象镜头：面具、袖口、灯光快速切换，不声称真实表演记录。

## 提示词模板

```text
观察式非遗纪录片镜头，主题是【非遗项目】。画面主体是【镜头主体】。
自然光，真实老作坊环境，保留工具磨损、粉尘、杂物和材料纹理。
手持摄影，有轻微微抖和焦点呼吸，不要丝滑广告片运镜。
低饱和、真实色彩、不过度美化。
镜头运动：【镜头运动】。
禁止：合成感人脸、塑料质感、过度干净、商业广告风、奇幻特效、错误器具、错误工艺步骤。
```

## 纪录片美学边界

AI 适合生成：

- 空镜。
- 工艺特写。
- 纹样动效。
- 历史复原意象。
- 器物、材料、空间、手部局部、背影、剪影。

必须真拍或用授权素材：

- 传承人正脸采访。
- 人脸特写、眼神、停顿、表情变化。
- 真实口述史。
- 历史照片、档案影像。
- 关键仪式、真实演出、真实工坊环境。

红线：

- 不用 AI 合成具体传承人的正脸身份。
- 不把 AI 生成内容冒充真实历史影像。
- 不生成错误工艺步骤并标注为真实流程。
- 不让 AI 质检替代人类专家审核。

## 手动即梦流程

```text
系统生成 Shot
→ 系统生成即梦提示词
→ 用户复制提示词到即梦官网
→ 用户设置比例、时长、参考图
→ 用户下载结果
→ 用户上传回系统
→ 系统绑定到 ShotAsset
→ 人工评分
```

记录字段：

```text
shot_id
heritage_topic
shot_type
prompt
jimeng_settings
generated_file
score_realism
score_ai_artifacts
score_craft_accuracy
score_material_texture
score_camera
score_editability
score_culture
failure_notes
decision
```

## 成功标准

15 个镜头里至少 9 个达到 4 分以上，且空镜、工艺特写、材料特写三类都稳定可用，纪录片方向进入 P1。

如果只有 5-8 个可用，纪录片方向调整为“AI 镜头辅助 + 真素材主导”。

如果少于 5 个可用，纪录片方向优先做策划、分镜、素材库和剪辑辅助，不押注 AI 视频生成。
