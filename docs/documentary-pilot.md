# 非遗纪录片 Pilot

## 目的

P0 不先做完整工程流水线，而是先验证即梦官网能否生成可用的非遗纪录片镜头。

执行方式：

```text
3 个主题（景德镇制瓷 / 苏绣 / 川剧变脸）
每个主题 5 个核心镜头
  establishing  空镜
  craft_close   工艺特写
  material_close 材料特写
  silhouette    剪影 / 背影
  imagery       意象镜头
总计 15 条核心镜头（≤ 15 条即梦提示词；触发真拍信号的镜头改人工拍摄）
用户复制到即梦官网手动生成
每条生成 1-3 个版本
下载回传
人工评分
总结可用镜头类型
```

> 实现侧：`server/agents/storyboard.py` 的 `SHOT_COUNT = 5` 是单一权威来源；fallback / LLM 路径都按 5 镜契约产出。当 brief 或 fact_cards 命中"传承人正脸 / 采访 / 口述史 / 现场演出"等信号时，silhouette 槽位会切换为 `portrait_interview` 并标 `requires_real_footage = true`，不生成即梦提示词。

## 15 个镜头

景德镇制瓷：

1. 老作坊外景空镜：清晨自然光，旧砖墙、陶土、木架、窑口烟气。
2. 拉坯手部特写：湿润瓷泥在转盘上成形，手指有泥水和细微颤动。
3. 修坯工具特写：金属工具轻刮瓷坯边缘，粉尘落下。
4. 青花绘制特写：毛笔蘸青料画纹样，瓷面有吸水痕迹。
5. 窑火意象镜头：火光、烟气、窑砖、器物轮廓，不出现具体人脸。

苏绣：

1. 老绣架空镜：窗边自然光，布面、线轴、木绷架、旧剪刀。
2. 针脚特写：针穿过绸缎，丝线反光，手部动作缓慢。
3. 理线镜头：不同颜色丝线被手指分开。
4. 纹样局部动效：花鸟纹样从针脚中逐渐显现，保持纪录片克制。
5. 工坊环境镜头：桌面略杂乱，布料、图稿、线团自然摆放。

川剧变脸：

1. 戏台后台空镜：旧木桌、脸谱、戏服、镜子、暖色灯。
2. 脸谱绘制特写：画笔在面具或脸谱纸样上落色，不做真人正脸。
3. 戏服纹理特写：刺绣、布料、肩甲、袖口随手移动。
4. 舞台剪影：演员背影或侧影甩袖，不展示可识别正脸。
5. 变脸意象镜头：面具、袖口、灯光快速切换，不声称真实表演记录。

## 提示词模板

```text
观察式非遗纪录片镜头，主题是【非遗项目】。画面主体是【镜头主体】。
自然光，真实老作坊环境，保留工具磨损、粉尘、杂物和材料纹理。
手持摄影，有轻微微抖和焦点呼吸，不要丝滑广告片运镜。
低饱和、真实色彩、不过度美化。
镜头运动：【镜头运动】。
禁止：合成感人脸、塑料质感、过度干净、商业广告风、奇幻特效、错误器具、错误工艺步骤。
```

## 纪录片美学边界

AI 适合生成：

- 空镜。
- 工艺特写。
- 纹样动效。
- 历史复原意象。
- 器物、材料、空间、手部局部、背影、剪影。

必须真拍或用授权素材：

- 传承人正脸采访。
- 人脸特写、眼神、停顿、表情变化。
- 真实口述史。
- 历史照片、档案影像。
- 关键仪式、真实演出、真实工坊环境。

红线：

- 不用 AI 合成具体传承人的正脸身份。
- 不把 AI 生成内容冒充真实历史影像。
- 不生成错误工艺步骤并标注为真实流程。
- 不让 AI 质检替代人类专家审核。

## 手动即梦流程

```text
系统生成 Shot
→ 系统生成即梦提示词
→ 用户复制提示词到即梦官网
→ 用户设置比例、时长、参考图
→ 用户下载结果
→ 用户上传回系统
→ 系统绑定到 ShotAsset
→ 人工评分
```

记录字段：

```text
shot_id
heritage_topic
shot_type
prompt
jimeng_settings
generated_file
score
failure_tags
decision
retry_count
notes
```

### 评分标准

采用 5 分制单一综合评分，附失败标签定位问题。

```text
5 分 — 可直接用于成片，不需要后期修
4 分 — 基本可用，需轻微后期（调色、裁剪）
3 分 — 有明显问题但素材有参考价值，不能直接用
2 分 — AI 痕迹严重或事实错误，不可用
1 分 — 完全不可用（画面崩坏、主体错误）
```

失败标签（多选）：

```text
ai_face         — 出现合成感人脸
plastic_texture — 塑料/油画质感
wrong_craft     — 工艺步骤或器具错误
ad_style        — 广告片风格，过度美化
motion_error    — 运镜不自然或物理错误
layout_error    — 构图/空间关系错误
irrelevant      — 画面主体偏离提示词
```

≥ 4 分为"可用"，用于统计成功率。

### 失败处理

```text
首次生成 ≤ 3 分：
  1. 分析 failure_tags，调整提示词中对应的约束
  2. 重新生成，最多重试 2 次
  3. 3 次均 ≤ 3 分 → 标记该镜头类型为"AI 不适合"

提示词调整规则：
  ai_face         → 加强"不出现人脸"约束，改用背影/手部/剪影
  plastic_texture → 加强"真实材质、磨损、粉尘"描述
  wrong_craft     → 补充具体工艺步骤描述，引用 FactCard
  ad_style        → 加强"手持、微抖、低饱和"约束
  motion_error    → 简化运镜描述，改为固定机位或缓慢平移
  layout_error    → 明确画面主体位置和比例
  irrelevant      → 精简提示词，减少干扰描述
```

## 成功标准

5 分制，≥ 4 分为可用。每个镜头最多 3 次生成机会（含重试）。以最终最高分计。

```text
15 个镜头中 ≥ 9 个可用，且空镜、工艺特写、材料特写三类各至少 2 个可用
  → 纪录片方向进入 P1

15 个镜头中 5-8 个可用
  → 分析可用镜头的类型分布
  → 纪录片方向调整为”AI 镜头辅助 + 真素材主导”

15 个镜头中 < 5 个可用
  → 纪录片方向不押注 AI 视频生成
  → 转向策划、分镜、素材库和剪辑辅助
```

阈值依据：15 镜覆盖 3 个主题 × 5 个镜头类型。9/15 = 60% 是最低可用率，低于此比例意味着每个主题平均不到 3 个可用镜头，无法拼出连贯叙事片段。三类各至少 2 个是为了验证 AI 在不同镜头类型上的稳定性，避免”只能拍空镜”的偏科问题。
