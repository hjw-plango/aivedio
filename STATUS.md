# 项目状态简报

## 当前结论

项目主线已经切换为：

```text
直接使用 waoowaoo-main 作为自己的 AI 影视生产底座
```

不再把根目录旧 Python 原型作为继续开发方向，也不在 P0 阶段融合 AIComicBuilder。

## 为什么这样定

代码分析后确认，`waoowaoo` 已经有一套可用的一致性基础：

- 角色档案。
- 角色别名。
- 多套角色外观。
- 外观变化原因。
- 角色参考图。
- 场景参考图。
- 当前选中场景图。
- clip / panel 级角色、场景、道具上下文。
- 摄影规则。
- 表演指导。
- 面板候选图和上一版图。

它不是完整状态机，但当前足够先进入真实测试。

## 已清理

- 文档主线改为 `waoowaoo-main`。
- 旧 Python 原型标记为历史参考。
- AIComicBuilder 标记为暂不融合。
- 完整项目记忆和连续性状态机从 P0 移除。
- Review、评分、即梦桥、纪录片固定审美从 P0 移除。
- 运行说明改为 `waoowaoo-main` 启动方式。
- 补充了测试内容准备清单。

## 当前测试入口

```text
RUNNING.md
docs/input-checklist.md
waoowaoo-main/README.md
```

## 下一步验证重点

真实测试时重点看：

- 本地环境能否顺利启动。
- 模型 API 配置是否顺畅。
- 角色和场景能否稳定生成。
- `CharacterAppearance` 是否足够承载状态变化。
- 分镜是否符合剧情。
- 面板图是否正确使用角色和场景参考。
- 视频链路是否能接上面板图。
- 计费关闭后是否仍有阻塞。

## 暂不处理

- 不删除旧目录。
- 不删除会员和计费代码。
- 不做无登录模式。
- 不做大规模重命名。
- 不做完整项目记忆系统。
- 不做 AIComicBuilder 迁移。
