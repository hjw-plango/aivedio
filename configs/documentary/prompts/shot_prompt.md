# 即梦视频提示词模板

## 使用说明

围栏内文本会被代码填入字段后复制到即梦。模板重点是可执行画面、参考图、一致性、声音与剪辑备注。

```text
纪录片第一章镜头。时间段:【chapter.timecode】。建议时长:【shot.duration】秒。画幅:16:9。

参考图使用:
【reference.manifest】

画面主体:
【shot.subject】

动作与状态:
【shot.action】

构图:
【shot.composition】

光线:
【shot.lighting】

运镜:
【shot.camera_motion】

旁白对应:
【shot.narration】

声音与剪辑备注:
【shot.sound_design】

事实细节:
【fact.detail】

统一风格:
【style.bible】

生成要求:
真实纪录片质感,4K,自然光,低饱和,保留灰尘、磨损、杂物、材料纹理和手部细节; 人物表情自然克制; 动作速度真实; 空间透视稳定; 不要广告片光泽。

负向约束:
塑料皮肤,过度磨皮,棚拍广告感,奇幻特效,错误工具,错误工艺动作,手指畸形,物体漂浮,过度运镜,字幕,水印,Logo,画面文字。
```
