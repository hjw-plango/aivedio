# 川剧变脸 · pilot 资料

> 来源:博物馆展板、公开非遗记录(整理稿,仅用于本项目验证 pipeline)

川剧变脸是川剧表演中以快速更换脸谱为标志的绝技,源于四川地区的戏曲传统。变脸主要分为抹脸、吹脸、扯脸三类技法,其中扯脸最为复杂,演员通过隐藏在头饰、衣襟之间的丝线,在转身瞬间扯下叠合的脸谱。

完整的变脸表演流程:

1. 化妆与脸谱准备:演员先在面部化底妆,再将多张丝绸或纸质脸谱按顺序叠合。
2. 戏服佩戴:外披戏服,袖口宽大,可遮掩动作。
3. 后台候场:演员调整呼吸,确认丝线连接与脸谱顺序。
4. 上台表演:在锣鼓与唱腔配合下,通过甩袖、转身、低头等动作触发脸谱切换。
5. 退场处置:演出结束,后台将脸谱拆下、清理、复位备用。

关键道具包括脸谱、戏服、丝线、化妆台、镜子。AI 镜头适合表现脸谱的细节特写、戏服布料的纹理、非特定演员甩袖的意象画面。真实传承人采访、真实演出片段、口述史必须真拍或使用授权档案。

需要避免:不能将 AI 合成画面伪装成真实表演记录,不能生成具体可识别的真实演员身份。
