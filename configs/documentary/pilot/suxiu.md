# 苏绣 · pilot 资料

> 来源:博物馆展板、公开非遗记录(整理稿,仅用于本项目验证 pipeline)

苏绣是流行于江苏苏州一带的传统刺绣,与湘绣、蜀绣、粤绣并称中国四大名绣。苏绣的代表针法为平针、套针、施针、虚实针等,其特点是图案秀丽、针脚细密、色彩雅致、丝光闪亮。

苏绣工艺流程为:

1. 选料:多采用塑形稳定的真丝绸缎做底料,丝线分为合股线、劈丝线两类。
2. 设计画稿:刺绣图案常取材于工笔花鸟、山水、人物。画稿先以墨线勾描在底料上。
3. 上绷:将绣片绷平绷紧在木质绷架上,以保证针脚均匀。
4. 理线:依据色彩深浅将丝线分股,理顺色阶,常一根丝可劈分为 16 至 32 丝。
5. 落针刺绣:按平针、套针等针法落线,绣面光泽随针向变化而呈现立体感。
6. 整熨:绣品完成后进行轻微熨烫,使丝面平整、无皱褶。

关键工具有绷架、剪刀、针、丝线轴、画稿。AI 镜头适合表现绣针穿过绸缎的特写、丝线的反光、纹样在针脚中逐渐成型的微距画面，也可以出现非特定绣娘或历史复原人物。真实传承人采访、口述仍需真拍或使用授权素材。
